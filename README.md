# Sistem Manajemen Event Kampus

## Deskripsi
Program untuk mengelola event kampus, peserta, panitia, dan pencarian event.

## Fitur
- Tambah Event
- Lihat Event
- Daftar Peserta
- Lihat Peserta
- Cari Event
- Hapus Event

## Struktur Data yang Digunakan
- Struct
- Class
- Linked List
- Queue

## Anggota Kelompok
1. Ilham Maulana (Ketua)
2. Nama Anggota 2
3. Nama Anggota 3
4. Nama Anggota 4

## Pembagian Tugas
- Ketua: Membuat kerangka project dan merge pull request.
- Anggota 2: Tambah Event dan Lihat Event.
- Anggota 3: Daftar Peserta dan Lihat Peserta.
- Anggota 4: Cari Event dan Hapus Event.
